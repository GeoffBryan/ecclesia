Project Name: ecclesia
Author: Geoff Bryan

Purpose: Allow users to convert text output files from the CATAPULT program to HDF format.  Provide four  python scripts (plt_contour_hdf, plt_scalar_hdf and plt_vector_hdf) and plt_stream_hdf) to plot the contents of these hdf files.

Directory structure:

|---> README.TXT
|
|---> bin          executable files and scripts
|
|---> data	   test data produced by CATAPULT
|
|---> install      installation scripts
|
|---> lib	   fortran libraries
|
|
|---> setup        python setup files
|
|
|---> src	   python and fortran source files
|
|---> tmp	   temporary directory for untracked files
